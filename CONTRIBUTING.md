# Contributing to Lịch Âm Việt Nam

Cảm ơn bạn đã quan tâm đến việc đóng góp cho dự án Lịch Âm Việt Nam!

## Cách đóng góp

1. Fork repository này
2. Tạo branch mới cho tính năng của bạn (`git checkout -b feature/AmazingFeature`)
3. Commit các thay đổi của bạn (`git commit -m 'Add some AmazingFeature'`)
4. Push lên branch (`git push origin feature/AmazingFeature`)
5. Mở một Pull Request

## Quy tắc đóng góp

- Viết code rõ ràng và có comment khi cần thiết
- Tuân thủ coding style hiện tại của dự án
- Thêm tests cho các tính năng mới nếu có thể
- Cập nhật documentation khi cần thiết

## Báo lỗi

Nếu bạn tìm thấy lỗi, vui lòng tạo một issue với:
- Mô tả chi tiết về lỗi
- Các bước để tái hiện lỗi
- Môi trường (platform, version, etc.)

## Đề xuất tính năng

Chúng tôi luôn hoan nghênh các đề xuất tính năng mới! Vui lòng tạo issue với tag `enhancement`.
